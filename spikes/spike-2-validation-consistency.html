Spike 2: Validation Consistency

Does Haiku return consistent rulings and valid JSON for the same inputs? Tests the exact system prompt from the discovery doc.

What this tests: We send 10 edge-case answers to Haiku multiple times each, using the exact system prompt planned for F5. We measure: (a) JSON parse success rate, (b) ruling consistency for identical inputs. Targets: >98% valid JSON, >95% consistent rulings.
